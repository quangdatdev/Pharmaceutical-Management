
#  Pharmaceutical-Management

Pharmacy sales desk processing application

## Technologies 

- Java Swing, JDBC, SQL Server

- JDK: 1.8 or later

## Features

- Inventory Management
- Prescription Management
- Billing and Invoicing
- Reporting and Analytics
